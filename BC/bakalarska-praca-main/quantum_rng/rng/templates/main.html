INTRODUCTION

Random numbers

Quantum RNG is an experimental project that demonstrates the generation of true quantum randomness using real quantum hardware. Instead of relying on traditional pseudo-random algorithms, this system obtains random values from the fundamental indeterminacy of quantum measurements.

Quatum Computer Used

The project currently uses the IQM Emerald system, a 54-qubit IQM Crystal Quantum Processing Unit built on superconducting transmon qubits. The qubits are arranged in a square lattice and interconnected through tunable couplers, enabling high connectivity and precise control of entanglement. The hardware supports arbitrary X and Y rotations as native single-qubit gates, while the CZ gate serves as its native two-qubit operation. This configuration provides a versatile and stable platform for generating quantum randomness through controlled measurements and entangled states.

How It Works

The generator leverages quantum superposition and measurement to produce unbiased random bits. Each value is derived from the collapse of a qubit’s state on an actual quantum processor — accessed through Amazon Braket. These results are then processed, stored in a local database, and made accessible via this web interface.
